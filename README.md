| **Tenzi** | `tenzi/` | Dice game simulation (`tenzi.py`) |

## Usage

Most analyses are Python scripts or Jupyter notebooks.

```bash
# Run a script
python games/tenzi/tenzi.py

# Open a notebook
jupyter notebook games/boggle/boggle.ipynb
```
